📈 通貨膨脹指數 (Inflation)

通膨指數表示貨幣價值下降、物價上漲的速度。 在班級經濟中：

計算方式：
通膨 = 現有資產 ÷ 基本資產
💡 現有資產越多 → 通膨越高；現有資產越少 → 經濟緊縮

例如：班上每人手上點數比基準多 → 市場錢多 → 通膨增加

💰 建議薪資

根據通膨指數調整每週薪資，計算公式：

新薪資 = 基本薪資 × 通膨指數 × 調整係數

📌 白話解釋：

  • 經濟好 → 薪資上升
  • 經濟差 → 薪資下降

🛒 消費者物價指數 (CPI)

反映商品和服務價格變化，計算公式：

CPI(%) = (目前價格 - 基準價格) ÷ 基準價格 × 100

📌 白話：

  • CPI 上升 → 物價漲 → 錢變薄
  • CPI 下降 → 物價穩定 → 錢值錢

💵 貨幣供給量 (Money Supply)

班級市場中流通的總點數：

貨幣供給量 = 每人點數 × 學生人數

📌 白話：

  • 貨幣多 → 消費力高 → 經濟活絡
  • 貨幣少 → 經濟緊縮 → 消費力下降

📊 GDP & 成長率

班級總經濟規模計算方式：

GDP = 本週薪資總和 + 貨幣供給量 × 0.35

成長率：

GDP成長率 = (本週GDP - 基準GDP) ÷ 基準GDP × 100%

📌 白話：

  • GDP 高 → 經濟活絡 → 班級資金充足
  • GDP 低 → 經濟冷清 → 班級資金不足
  • 成長率正 → 經濟成長，負 → 經濟縮水

📈 股票建議變動 (stockChange)

綜合指標公式：

stockChange (%) = (GDP成長率*4 + 貨幣增長率*3 + CPI/100 - (inflation-1)*3) * 100
限制：-10% ≤ stockChange ≤ 10%

📌 白話：

  • GDP 成長 → 股票上漲
  • 貨幣多 → 股票上漲
  • CPI 輕微上漲 → 股票小幅上漲
  • 通膨過高 → 壓低股票
  • 漲跌限制 ±10% → 防止波動太大
